# MängelLog

> Kaution sichern: Fotos, Mängel & Zeitstempel. 100% offline, kein Abo.

## Problem

Vermieter behält Kaution weil Mängel beim Einzug nicht beweisbar dokumentiert wurden.

## Lösung (kostenlos)

Unbegrenzte Mängeleinträge mit Foto, Zeitstempel, Zählerstand, Schlüsselprotokoll.

## Premium freischalten

Nutzer will PDF für Vermieter/Hausverwaltung exportieren.

Einmalige Zahlung, kein Abo. Preis wird live aus dem Play Store geladen.
